# Card Type Symbols

SVG icons for MTG card types, sourced from the [MTG Wiki](https://mtg.fandom.com/wiki/Card_type).

## Files

| File | Card Type |
|------|-----------|
| `artifact.svg` | Artifact |
| `creature.svg` | Creature |
| `enchantment.svg` | Enchantment |
| `instant.svg` | Instant |
| `land.svg` | Land |
| `multi-type.svg` | Cards with multiple types |
| `planeswalker.svg` | Planeswalker |
| `sorcery.svg` | Sorcery |

## Types without symbols

Kindred and supplemental types (Conspiracy, Dungeon, Emblem, Hero, Phenomenon, Plane, Scheme, Vanguard) do not have icons. Fall back to text for these.
